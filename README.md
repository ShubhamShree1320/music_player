# Shubham's Music_Player

To run this project, follow these steps:

1. Clone the repo.
2. Navigate into the `music_player` directory:
   ```bash
   cd music_player
    npm init -y
    npm install
    tsc --init
    ```
3. Open the folder and start `index.html`.
